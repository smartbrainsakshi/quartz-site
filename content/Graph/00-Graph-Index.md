# Graph Algorithms Notes

[← Back to All Topics](../index.md)

Welcome to my personal knowledge base for **Graph Algorithms**, built while working through [Striver's Graph Series](https://www.youtube.com/playlist?list=PLgUwDviBIf0oE3gA41TKO2H5bHpPd7fzn) by take U forward.

Every note follows a consistent format: **intuition first, then visuals, then code (Java + Python), then complexity, then pitfalls** — so you can revise a whole algorithm in minutes, not hours.

## Topics

### [01 · Foundations](01-Foundations/00-Foundations-Index.md)
Graph representations, types, BFS & DFS traversal basics.

### [02 · BFS / DFS Problems](02-BFS-DFS-Problems/00-BFS-DFS-Index.md)
Connected components, cycle detection, bipartite checks, flood fill, rotting oranges, and more — all built on top of BFS/DFS.

### [03 · Topological Sort](03-Topological-Sort/00-Topological-Sort-Index.md)
Kahn's algorithm, DFS-based topo sort, cycle detection in directed graphs, course schedule problems.

### [04 · Shortest Path Algorithms](04-Shortest-Path-Algorithms/00-Shortest-Path-Index.md)
Dijkstra, Bellman-Ford, Floyd-Warshall, and shortest path problems on weighted/unweighted graphs.

### [05 · MST & Disjoint Set](05-MST-and-Disjoint-Set/00-MST-Disjoint-Set-Index.md)
Prim's, Kruskal's, Disjoint Set Union (Union by Rank/Size + Path Compression), and DSU-based problems.

### [06 · Other Algorithms](06-Other-Algorithms/00-Other-Algorithms-Index.md)
Bridges, articulation points, Kosaraju's & Tarjan's SCC algorithms.

### [07 · Sliding Window & Two Pointers](07-Sliding-Window-and-Two-Pointers/00-Sliding-Window-Index.md)
A separate DSA topic (not part of the Graph Series) — all 4 core patterns: constant window, longest window, count windows, shortest window.
